# Ecommerce Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing eCommerce sales data to identify business trends, customer behavior, product performance, and regional sales insights. The dashboard provides interactive visualizations and KPI tracking to help businesses make data-driven decisions.

The analysis was created using Microsoft Excel with Pivot Tables, KPIs, charts, and dashboard visualizations.

---

## Objectives

* Analyze overall sales performance.
* Identify top-performing products and categories.
* Monitor yearly sales growth.
* Compare regional sales performance.
* Track customer segments and shipping trends.
* Generate actionable business insights using data visualization.

---

## Dataset Information

The dataset contains eCommerce order records with the following information:

* Order ID
* Order Date
* Ship Date
* Customer Details
* Product Category & Sub-Category
* Sales
* Quantity
* Profit
* Region
* Shipping Mode

### Dataset Features

* Multi-year sales data
* Customer segmentation analysis
* Product performance tracking
* Regional sales comparison
* Profitability analysis
